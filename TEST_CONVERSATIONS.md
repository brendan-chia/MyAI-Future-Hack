# How to Use These for Testing

## Test with `/start` mode (Batch Analysis):

1. Send `/start` or `/begin`
2. Copy-paste one of the conversation threads above
3. Send multiple messages from the conversation as separate messages
4. Send `/analyze` or `/scan` when done
5. Bot should detect the scam pattern and ask clarification questions if confidence is borderline

## Expected Bot Behavior:

- **Investment Scam**: Should detect HIGH_RISK (upfront payment, unrealistic returns)
- **Loan Scam**: Should detect HIGH_RISK (upfront fee before loan, no CCRIS check)
- **Love Scam**: Should detect HIGH_RISK (romance + money request)
- **Job Scam**: Should detect HIGH_RISK (upfront training fee, too good to be true)
- **Parcel Scam**: Should detect HIGH_RISK (unexpected package + urgent payment)
- **Macau Scam**: Should detect HIGH_RISK (unsolicited prize + deposit required)
- **Mixed Thread**: Should detect HIGH_RISK with confidence boost (multiple escalation tactics)

## For Clarification Testing:

If confidence is MEDIUM (60-70%), bot should ask questions like:
- "Did you sign up for this intentionally?"
- "Are they promising unrealistic high returns?"
- "Did you actually apply for this loan?"
- etc.

Depending on answers, verdict will refine to HIGH or LOW risk.
